怎样使用surge静态托管

2024-09-05

1.安装 Surge：
首先，确保你的系统上安装了 Node.js。然后，win+R,输入cmd，回车，输入 Surge：npm install --global surge(下载node.js:https://nodejs.org/zh-cn)

完成后大概就是下面这样子的了

C:\Users\xiany\AppData\Roaming\npm\surge ->

C:\Users\xiany\AppData\Roaming\npm\node_modules\surge\lib\cli.js

  • surge@0.20.4

updated 1 package in 28.177s

2.部署网站：
win+R，输入cmd回车，输入surge
第一次开始大概是这个样子的

Welcome to Surge! (surge.sh)

Login (or create surge account) by entering email & password.

email: 自己输入你的邮箱

password: 自己输入你的密码

Running as 你的邮箱地址 (Student)

project: (这里会自动显示你的项目地址)(可以直接将项目拖入进去)

domain：*****.surge.sh   注意这里的****需要通过键盘上的左右键移动光标删除输入想要的域名前缀

upload: [====================] 100% eta: 0.0s (127 files, 5275869 bytes)

CDN: [====================] 100%

IP:
现在我们打开浏览器，在地址栏输入******.surge.sh  回车就可以看到效果了

3.更新
更新直接win+R,输入cmd，回车，输入 Surge
project： 拖入更新完成的文件，回车

4.如何退出当前的邮箱 ：win+R,输入cmd，回车，输入：surge logout，回车
